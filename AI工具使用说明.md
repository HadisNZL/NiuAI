# AI 工具调用功能说明

## 概述

NIU AI 现在支持自动调用工具，AI 可以自主执行文件操作、Git 操作和项目分析。

## 可用工具

### 1. 文件操作

**read_file** - 读取文件内容
- 示例对话：
  - "帮我看看 app.py 的内容"
  - "读取 README.md"
  - "分析一下 config.py 做了什么"

**write_file** - 写入或创建文件
- 示例对话：
  - "创建一个 test.py 文件，写入 print('hello')"
  - "修改 config.py，添加..."

### 2. 项目分析

**get_project_structure** - 获取项目结构
- 示例对话：
  - "这个项目有哪些文件？"
  - "帮我看看项目结构"
  - "列出所有的函数和类"

### 3. Git 操作

**git_status** - 查看 Git 状态
- 示例对话：
  - "git status"
  - "查看当前有哪些更改"
  - "有什么文件改动了？"

**git_diff** - 查看代码差异
- 示例对话：
  - "git diff"
  - "看看我改了什么"

**git_commit** - 提交更改
- 示例对话：
  - "提交这些更改，消息是'修复bug'"
  - "commit 这些文件"

## 工作原理

1. 用户发送请求（如"帮我读取 app.py"）
2. AI 识别需要调用工具
3. 自动调用 `read_file` 工具
4. 获取文件内容后，AI 基于内容生成回复
5. 用户看到完整的分析结果

## 界面显示

对话中会显示：
- 🔧 调用工具：显示 AI 正在调用哪个工具
- ✅ 工具结果：显示工具执行结果（简略）
- AI 最终基于工具结果生成的回答

## 注意事项

1. **权限控制**：只能访问项目目录下的文件
2. **安全性**：危险操作（如删除文件）需要明确确认
3. **模型支持**：需要模型支持 Function Calling（如 GPT-4、DeepSeek等）

